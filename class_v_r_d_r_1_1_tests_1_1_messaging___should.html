void VRDR.Tests.Messaging_Should.ValidateMessageHeader ( )
inline

Tests the validation of the message header.

This test parses a message from a JSON fixture and validates its header. It ensures that the message is of type DeathRecordSubmissionMessage. It also verifies that a MessageRuleException is thrown when the message header is invalid, and checks that the exception message and source message type are correct.

Exceptions
MessageRuleExceptionThrown when the message header validation fails due to the certificate number being more than 6 digits long.
